# README

Esse repositório contém exercícios básicos.

No diretório de test, você não pode modificar. VocÊ só pode modificar o conteúdo do diretório de lib (no qual o código será implementado).

Para executar o conjunto de testes basta executar:

```shell
dart test
```

A medida que você vai implementando as funções, a ideia que os testes fiquem verde e passem.
